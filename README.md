# Codeforces Random Unsolved

Get a random Codeforces problem a user hasn't solved, filtered by tags and rating. No login needed.

## How to Use

- Enter a handle (e.g., `tourist`)
- Select tags
- Pick rating range
- Get a random unsolved problem

## Tech

- Node.js + Express
- HTML, CSS, JS
- Uses Codeforces API

## Setup

```bash
git clone <this-repo>
cd cf-random-unsolved
npm install
npm start
```
